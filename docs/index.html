Intelligence Modules
8 modules covering the full OSINT attack surface
Each module operates independently or as part of a full reconnaissance workflow. All data collection is passive or minimally intrusive. No exploitation capabilities.
01
Domain Intelligence
DNS records (A, AAAA, MX, NS, TXT, SOA, CNAME, SRV), WHOIS data, HTTP headers, security headers scoring, robots.txt analysis, sitemap enumeration, redirect chain tracking.
DNSWHOISHTTP
02
SSL/TLS Analysis
Certificate subject, issuer, SAN entries, serial number, validity period, expiry countdown, protocol version, cipher suite, key exchange strength.
x509TLSexpiry
03
Subdomain Enumeration
Passive discovery via Certificate Transparency logs (crt.sh). Active DNS brute-force with 80+ common subdomain wordlist. Concurrent resolution with configurable thread pool.
CT logsDNS brutethreaded
04
IP Intelligence
IPv4/IPv6 classification, reverse DNS, geolocation (country, region, city, coordinates), ISP identification, organization, ASN lookup, private/global/multicast detection.
GeoIPASNrDNS
05
Email Intelligence
MX record enumeration, email provider identification (Google, Microsoft, ProtonMail, Zoho), SPF/DMARC/DKIM validation, DKIM selector discovery (10 common selectors).
MXSPFDMARCDKIM
06
Technology Fingerprinting
Detection of 24+ technologies: CMS (WordPress, Drupal, Joomla, Shopify), frameworks (React, Angular, Vue), CDN (Cloudflare, AWS), analytics, CAPTCHA, server software.
CMSframeworksCDN
07
Port Scanner
Lightweight TCP connect scanner for 23 common service ports. Concurrent scanning with configurable thread pool and timeout. Service identification (SSH, HTTP, SMB, RDP, databases).
TCP23 portsthreaded
08
Social Media Discovery
Username enumeration across 15 platforms: GitHub, Twitter/X, Instagram, LinkedIn, Reddit, TikTok, YouTube, Pinterest, Medium, Twitch, Keybase, HackerOne, Bugcrowd, Dev.to, Mastodon.
15 platformsthreaded
Deployment
On-premises installation — zero cloud dependencies
PublicEye runs entirely on your infrastructure. No data leaves your environment. No API keys required for core functionality. Optional integrations available for extended capabilities.
# Install git clone https://github.com/SiteQ8/PublicEye.git cd PublicEye pip install -r requirements.txt # Full scan on a domain python3 publiceye.py -t example.com -m full # Single module python3 publiceye.py -t example.com -m ssl # Multiple modules python3 publiceye.py -t example.com -m domain,ssl,subdomains # IP reconnaissance python3 publiceye.py -t 8.8.8.8 -m ip # Social media username check python3 publiceye.py -t johndoe -m social # HTML report output python3 publiceye.py -t example.com -m full -o html
